# RangeCrawler

**Realistic, headless user simulation framework for cyber ranges, training, and red/blue/purple exercises**

Currently in early planning & documentation phase.

[![License: GPL v3](https://img.shields.io/badge/License-GPL%20v3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)
[![Project Status: Planning](https://img.shields.io/badge/status-planning-yellow.svg)]()
[![Contributions Welcome](https://img.shields.io/badge/contributions-welcome-brightgreen.svg)]()

## Features (planned)
- Fully headless agents (Linux, Windows, macOS nice-to-have)
- Modern web dashboard for scenario creation and live monitoring
- Timeline-based user behavior engine
- Docker-first deployment (multi-arch)
- Strong environment health & usability checks
- 100% open source under GPL-3.0
